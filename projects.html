Composer iPad App

Tufts Course: Comp 150: Music Apps on the iPad

Details: This was a group project that lasted one semester. This app was meant to be used by composers who can easily check the music they are writing.

Features:
1) Allows users to press on the staff drawn in the app and notes will appear where they pressed it.
2) The user can use different tap gestures to add different kind of notes: whole, half, quarter, etc.
3) There are many different instruments that the notes can be played with and the app will play the notes all together when the user chooses to listen to their composition.

Java 2D Space Game

Language: Java

Details: This is a Java game that is based on real time strategy and civilization games. (Swing GUI, Draw2d, inheritance, abstract classes, fileIO, etc.)

Features: Instructions are shown on the main screen.
1) This game is a space race game to colonize all the possible planets.
2) The player can set their own name, color, and starting wealth, along with number of planets, and number of enemies.
3) Player wins by colonizing all planets.
4) There are three types of ships.
5) Fighter ships: fast, fights well, and cost the least, can destroy other fighter ships.
5) Mother ships: mediums speed, doesn't fight well, can help give shield to other friendly ships it passes by.
6) Warships: largest ships, slow, has lots of damage, can colonize 2 planets.
7) Players start with 1 warship and 1 fighter ship.
8) Time on the status shown on the top gives when the year will advance and when the income will be collected.
9) Many other features described in source folder readme in the GitHub Repo...

View game repo on GitHub
